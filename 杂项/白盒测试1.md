### 2、阅读下面这段程序，使用逻辑覆盖法进行测试，请问哪一组关于(a,b,c)的输入值可以达到条件覆盖。
```c
int func(int a, int b, int c)
{
    int k = 1;
    if ((a > 0 || (b < 0) || (a + c) > 0))
        k = k + a;
    else
        k = k + b;
    if (c > 0)
        k = k + c;
    return k;
}
```

- A: (a,b,c)=(3,6,1)、(-4,-5,7)
- **B: (a,b,c)=(2,5,8)、(-4,-9,-5)**
- C: (a,b,c)=(6,8,-2)、(1,5,4)
- D: (a,b,c)=(4,9,-2)、(-4,8,3)
